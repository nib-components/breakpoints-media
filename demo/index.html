This demo utilises
  • The grid classes for layout. The grid will also define a 'breakpoint' value to the html:before element
  • The breakpoints JS module to read the 'content' value of the html:before element on window resize
  • The body-attributes JS module to get the current breakpoint value and add it to a body data-attribute
